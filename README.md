# Midpoint Activity Finder

This is a code bundle for Midpoint Activity Finder. The original project is available at https://www.figma.com/design/gemfN7o30OczBMK1tlnfxr/Midpoint-Activity-Finder.

## Running the code

Run `npm i` to install the dependencies.

Run `npm run dev` to start the development server.

Click on the `backend` and `midpoint-native` directory paths to understand how to set up your backend and frontend.
